Web Design

Dans le processus de création des sites web, le Web Design est aujourd’hui une spécialité à part entière.C'est un passage obligé d’une conception internet réussie.

Web Development

Le développement Web est le codage ou la programmation qui permet de faire fonctionner un site Web, selon les exigences du propriétaire. Il traite principalement de l’aspect non visuel de la création de sites Web, qui comprend le codage et l’écriture du site en question.
